# BankingAppUI
Banking App UI to practice in Jetpack Compose components
<br>

## Features
- Dynamic theme changing due to device theme (Dark/Light) <br>
- Dynamic color scheme changing due to device wallpaper <br>

## Technologies Used
- Jetpack Compose <br>
